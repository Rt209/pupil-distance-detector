# pupil-distance-detector

A modular OpenCV project for classical pupil detection and inter-pupil distance measurement. The repository is designed for GitHub portfolio presentation and emphasizes reusable modules, composable pipelines, and clear separation of concerns.

## Features

- Detects left and right pupil regions from an input face image
- Draws circles, contours, and labels on the output image
- Computes left/right pupil centers and the pixel distance between them
- Supports multiple interchangeable classical image-processing pipelines
- Keeps each image-processing technique isolated in its own reusable module

## Installation

```bash
pip install -e .
```

## Image Storage

Use the `data/` folder to organize images:

- `data/input/`: your own input face images for processing
- `data/output/`: annotated output images generated by the CLI
- `data/samples/`: optional demo images for documentation or quick testing

## Usage

```bash
python -m pupil_distance_detector.main ^
  --input data/input/face.jpg ^
  --output data/output/annotated_v1.jpg ^
  --pipeline v1
```

### CLI Arguments

- `--input`: path to the input face image
- `--output`: path for the annotated output image
- `--pipeline`: one of `v1`, `v2`, or `v3`
- `--grayscale`: treat the input image as grayscale

## Pipelines

### Pipeline V1

`Gaussian Blur -> Canny -> Contour -> Candidate Filter -> Center -> Distance`

Best for images with strong edge contrast around the iris and eyelids.

### Pipeline V2

`Gaussian Blur -> Binarization -> Morphology -> Hough Circle -> Candidate Filter -> Center -> Distance`

Best for relatively frontal faces with clear dark pupil contrast.

### Pipeline V3

`Perspective -> Blur -> Sobel -> Contour -> Morphology -> Candidate Filter -> Center -> Distance`

Best for slightly skewed images where eye alignment benefits from a rectified processing region.

## Output Example

The program returns:

- Annotated image with pupil boundaries and the connecting distance line
- Left and right pupil center coordinates `(x1, y1)` and `(x2, y2)`
- Inter-pupil distance in pixels

Example terminal output:

```text
Pipeline: v2
Left pupil center: (142, 108)
Right pupil center: (243, 110)
Distance: 101.02 px
Saved annotated output to data/output/annotated_v2.jpg
```

## Validation Suggestions

- Test with multiple images under different lighting conditions
- Prefer frontal or near-frontal faces for the strongest classical-processing performance
- Confirm the two selected candidates are approximately horizontal and near the upper-middle face region

## Notes

- This repository intentionally avoids deep learning and relies only on classical image-processing methods.
- The default candidate selection logic assumes the input image is a face crop or a reasonably centered face portrait.
- Perspective rectification is implemented as a reusable module and applied through a configurable quadrilateral warp.
